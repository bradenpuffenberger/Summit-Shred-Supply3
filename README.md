# Summit-Shred-Supply
Ecommerce website for school project
The goal of this project is to create a website where people can log in to an account, post items for sale and buy as well.

## Auth

This repo uses AWS Amplify Gen 2 with Cognito email authentication. See `COGNITO_SETUP.md` for local sandbox and Amplify Hosting setup.

Useful commands:

```powershell
npm ci
npm run sandbox
```
